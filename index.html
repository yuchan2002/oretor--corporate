I — Invitation

情報が飽和する時代、
意思決定は芸術に近づく。

このリトリートは、データと直観を往復しながら「次の一手」を再設計したい経営層のための、1日完結型の招待制プログラムです。主宰者が独自に開発した分析フレームワークを、少人数・対話形式で共有します。

  • 年商1億円以上の経営者・創業者
  • 上場企業の役員およびC-suite
  • 独立系プロフェッショナル・投資家
  • 事業承継・次世代経営を担う方
  • AI・データ活用の実装を探索する方

選考について

本プログラムは応募制・選考制です。お申込後、事務局にて参加意図・ご経歴を確認のうえ、ご案内を差し上げます。参加者の機密保持と空気感を保つため、少人数・同水準の方々のみで構成します。

II — Program

1日を、濃密に設計する。

朝から夜まで、9時間の連続したセッション。講義・対話・個別相談・会食を織り交ぜ、参加者全員が「自社に戻った翌日から変えられる何か」を持ち帰るよう設計しています。

10:00
Welcome Reception
参加者同士の自己紹介と、1日の目的共有。ウェルカムドリンクとともに、静かに1日を始めます。
10:30
Session I — AIと経営の交差点
生成AI・機械学習の現在地と、経営における実装の解像度。ケーススタディを交えた90分の講義＋対話形式セッション。
12:30
Executive Lunch
シェフ監修のコースランチ。参加者同士の対話のための時間として設計しています。
14:00
Session II — データドリブン意思決定の構造
データを「見る」から「判断する」へ。独自の分析フレームワークを、参加者の課題に適用しながら解説します。
16:00
Session III — Private Consultation
参加者1名ごとに90分の個別セッション。自社の具体的な課題について、1対1で深く掘り下げます。
17:30
Networking Reception
同水準の経営者同士による、静かで深い対話の時間。一生のつながりが生まれる場として設計しています。
19:00
Private Dinner
1日の締めくくり。オフレコで語られる本音の対話を通じ、リトリートを完結させます。

VI — Inquiries

よくあるご質問

参加はどのように決定されますか？
お申込フォームでいただいたご経歴・参加意図を事務局にて確認のうえ、ご案内を差し上げます。選考結果は3営業日以内にご連絡いたします。同水準の参加者で構成するための選考制であり、ご返金事由には該当しません（不参加の場合、お支払いは発生しません）。
支払いはどのように行いますか？
選考通過後、事務局より個別にStripe請求書をメール送付いたします。クレジットカード（VISA / Mastercard / AMEX / JCB）での1回払いに対応しております。銀行振込をご希望の方は個別にご相談ください。
キャンセルポリシーを教えてください。
開催21日前まで：全額返金 / 開催14日前まで：50%返金 / 開催7日前まで：25%返金 / それ以降：返金不可。やむを得ない場合は次回開催への振替をご相談いただけます。
服装の指定はありますか？
Smart Casualを推奨しております。ディナーセッションも含め、リラックスできる装いでご参加ください。
機密保持について教えてください。
参加者全員に守秘義務に関する同意書へのサインをお願いしています。セッション内で共有された個別事業情報は、事務局および参加者間で厳密に保護されます。
